# PROYECTO-3ER-P-IAW
mi repositorio acerca del 3er parcial materia: IAW
Elevator Pitch (Descripción rápida):
GradeSort ORGANIZA Y CELEBRA POR LOS 10!!!!

El Problema (Justificación / El "Porqué"):
Muchos estudiantes tienen dificultades para mantener un registro ordenado de sus calificaciones, especialmente cuando cursan varias materias o carreras. Esto puede generar confusión y dificultar la evaluación de su progreso académico.Elegimos este proyecto porque detectamos la necesidad de una herramienta práctica y automatizada que facilite la organización de calificaciones, especialmente útil para alumnos que manejan grandes volúmenes de datos o buscan mejorar su rendimiento mediante un análisis claro y visual.

Público Objetivo:
La aplicación está dirigida principalmente a estudiantes de preparatoria y universidad, así como a docentes que deseen gestionar y comparar calificaciones de forma rápida y eficiente.
